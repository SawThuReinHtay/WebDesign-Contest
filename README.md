A website developed for a **Web Design Competition**, focusing on supporting educational development in rural areas of Myanmar.  

The website was designed for a fictional **Non-Governmental Organization (NGO)** whose mission is to improve education access by building schools, supporting libraries, and providing learning resources to underprivileged communities.

This project focuses on **modern web design principles, clear information presentation, and responsive layout** to effectively communicate the NGO's mission and encourage public support.

---

## Project Overview

The purpose of the website is to:

- Raise awareness about education inequality in rural areas
- Showcase the NGO’s mission and projects
- Encourage donations and community participation
- Provide information about educational initiatives
- Present a professional and accessible interface for visitors

The project was developed as part of a **web design contest**, where it achieved **2nd place**.

---

## Key Features

- Clean and modern website layout
- Responsive design for mobile, tablet, and desktop
- Informational pages about the NGO's mission and projects
- Sections highlighting rural education challenges
- Donation awareness and support information
- Simple and intuitive navigation
- Visual storytelling through images and structured content
